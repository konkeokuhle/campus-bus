# Campus Bus Tracking System

A real-time bus tracking web application for students and drivers to manage campus-residence transportation.

## Features

-  Real-time bus tracking on interactive map
-  User authentication (Students, Drivers, Admins)
-  Live location updates every 5 seconds
-  ETA calculations for each stop
-  QR code check-in for boarding
-  Voice alerts for bus arrivals
-  Lost and found system
-  Favorite routes and notifications

## Technology Stack

- **Backend:** Python Flask
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Maps:** Leaflet.js with OpenStreetMap
- **Real-time:** Socket.IO
- **Authentication:** Flask-Login, JWT
